Aprendendo:

HTML

Atualmente, estou estudando html e irei apresentar um site com html e css para o dia 25/04.

HTML é a linguagem de estilo responsável por estruturar o conteúdo de páginas na web. Com o HTML, organizamos elementos como títulos, parágrafos, imagens, links, botões, listas e muito mais, criando a base para qualquer site ou aplicação web.

Css

Atualmente, estou estudando css e irei apresentar um site com html e css para o dia 25/04

CSS é a linguagem de estilo usada para definir o visual e o design de uma página web. Com o CSS, é possível estilizar elementos HTML, controlando cores, tamanhos, fontes, espaçamentos, posicionamentos, animações e muito mais.

bootstrap

Também estou estudando bootstrap, que irei utilizar para deixar meu site responsivo.

Bootstrap é um framework front-end que facilita a criação de sites modernos, bonitos e responsivos com muito mais agilidade. Ele oferece uma coleção pronta de códigos CSS, JavaScript e componentes HTML reutilizáveis — como botões, formulários, modais, carrosséis e barras de navegação — que ajudam a padronizar e acelerar o desenvolvimento web.

JavaScript

Estou estudando javascript, mas não estou utilizando neste projeto.

JavaScript é uma linguagem de programação amplamente utilizada para criar interatividade e dinamismo em páginas web. Com o JavaScript, é possível adicionar animações, manipular elementos HTML e CSS, validar formulários, realizar requisições assíncronas, criar jogos e muito mais. É uma das principais linguagens do desenvolvimento web moderno.
